# Safe-All
The Project 
Safe for All 
Γειά σας, είμαστε η ομάδα Safe for All του 10ου Δημοτικού Σχολείου Ξάνθης και συμμετέχουμε στον διαγωνισμό ανοικτών τεχνολογιών με ένα έργο το οποίο μπορεί να βοηθήσει όλους τους ανθρώπους να ζήσουν με μεγαλύτερη ασφάλεια. Θέλουμε να δημιουργήσουμε ένα έξυπνο ασφαλές αυτοκίνητο με σκοπό να περιορίσουμε στο ελάχιστο τα τροχαία ατυχήματα. 
Παρατηρήσαμε ότι στην πόλη μας την Ξάνθη τον τελευταίο καιρό υπήρξαν πολλά τροχαία ατυχήματα ακόμη και στο κέντρο της πόλης. Μελετήσαμε το φαινόμενο και ήρθαμε σε επαφή τόσο με την τροχαία Ξάνθης όσο και με τον πατέρα παιδιού της ομάδας μας ο οποίος είναι ιδιοκτήτης εταιρίας οδικής βοήθειας. 
Μας περιέγραψαν την όλη κατάσταση και εμείς θελήσαμε να προτείνουμε λύση μέσω των ανοικτών τεχνολογιών. Θέλουμε να δημιουργήσουμε μια μακέτα αυτοκινήτου το οποίο θα εμπεριέχει πολλά συστήματα ασφαλείας: αισθητήρες τρακαρίσματος, αυτόματα φρένα, αυτόματες πόρτες, αυτόματα φώτα, αυτόματες ζώνες ασφαλείας, έλεγχος ελαστικών, έλεγχος πτώσης – τρακαρίσματος, 
θα εκπέμπει σήματα σε περίπτωση κινδύνου και πολλά άλλα. Είμαστε σίγουροι ότι το αυτοκίνητό μας θα δράσει στην κοινωνία της Ξάνθης με δύο τρόπους: 1. θα εξασφαλίσει ασφαλείς μετακινήσει και 2. θα αποτελέσει πλατφόρμα εκπαίδευσης σχετικά με την ασφαλή οδήγηση μιας και θα θέλαμε το έξυπνο αυτοκίνητό μας να είναι διαθέσιμο και για εκπαιδευτικούς λόγους. 

Υλικά για το έργο μας:
	Πλακέτες Micro: Bit (X3)
	Καλώδια – αισθητήρες – κινητήρες για  Micro: Bit
	Υλικά για μακέτα: χαρτόνια, στικ ξύλου και αναλώσιμα 
	Δομικά μέρη τύπου lego και Hub που να παίρνει Micro: Bit
